# neuroformats-rs
Handling of structural neuroimaging file formats for [Rust](https://www.rust-lang.org/).

This is work in progress, come back another day.


## Usage example

```rust
use neuroformats::read_curv;
curv = read_curv("path/to/lh.thickness")
```
